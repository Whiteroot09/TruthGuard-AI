About the Project

TruthGuard AI – Unmask the Fake. Trust What’s Real.

TruthGuard AI is a web application that integrates machine learning models in its backend to perform real-time verification of textual and visual content. It analyzes news articles to detect misinformation and uses deepfake detection techniques to assess the authenticity of uploaded images, offering users a reliable tool against digital deception.

Motivation behind the Project:

In a world where misinformation moves fast, TruthGuard AI empowers users to separate fact from fiction. Powered by cutting-edge machine learning, it verifies news and detects image manipulation—ensuring you engage only with authentic content.

Project Goals:

  • To provide a user-friendly interface for news and image verification.
  • To educate users about the importance of verifying information.
  • To contribute to the fight against misinformation and digital manipulation.

Key Features:

  • Analyzing news articles with various machine learning algorithms to predict their authenticity.
  • Analyzing image with various machine learning algorithms to predict their authenticity.

Algorithms Used

  • Decision Tree
  • Random Forest
  • Gradient Boosting
  • Logistic Regression
  • Neural Network
  • Convolutional Neural Network (CNN) (for image analysis)

Purpose of use multiple Algorithms

Using multiple algorithms allows us to leverage the strengths of each model, improving overall accuracy and robustness. Ensemble methods combine predictions from different models to achieve better performance than any single model alone.

Performance Precision

  • Text Analysis: Up to 96% accuracy in detecting misinformation.
  • Image Analysis: High precision in identifying deepfakes and manipulated images.
